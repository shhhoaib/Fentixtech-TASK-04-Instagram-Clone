# Instagram Profile UI Clone

A responsive Instagram Profile UI clone built with HTML & CSS, featuring 3 interactive screens showcasing modern web design principles.

## 🎯 Project Overview

This project recreates the Instagram profile interface with a focus on:
- **Flexbox & CSS Grid** layouts for responsive design
- **Semantic HTML** structure
- **Modern CSS** styling with gradients and animations
- **3 Interactive Screens:**
  1. **Profile Screen** - User profile with stats and post grid
  2. **Stories & Highlights** - User stories and highlighted content
  3. **Feed Screen** - Social feed with multiple posts

## 📁 Folder Structure

```
├── index.html          # Main HTML file with all 3 screens
├── css/
│   └── styles.css      # Complete stylesheet
├── assets/             # Assets folder (for future images)
└── README.md           # Project documentation
```

## ✨ Features

### Screen 1: Profile
- User avatar and profile information
- Stats display (Posts, Followers, Following)
- User bio with links
- Follow and Message action buttons
- Tab navigation (Posts, Reels, Saved, Tagged)
- 3x3 grid of user posts with hover effects

### Screen 2: Stories & Highlights
- Story carousel with scrollable content
- Highlights section with custom categories
- Smooth hover animations
- Interactive elements

### Screen 3: Feed
- Multiple feed posts with user info
- Post header with user avatar and details
- Post image showcase
- Action buttons (Like, Comment, Share, Save)
- Post captions with clickable usernames
- Responsive design for mobile viewing

## 🛠️ Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, Grid, Gradients, Animations
- **Vanilla JavaScript** - Screen navigation

## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/shhhoaib/Fentixtech-TASK-04-Instagram-Clone.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Fentixtech-TASK-04-Instagram-Clone
   ```

3. Open `index.html` in your web browser:
   - Right-click on `index.html` → Open with Browser
   - Or use a live server extension (VS Code Live Server)

4. Navigate between screens using the bottom navigation buttons:
   - 🏠 Home - Profile Screen
   - 👤 Profile - Stories & Highlights
   - 🔍, ➕, ❤️ - Navigate between screens
